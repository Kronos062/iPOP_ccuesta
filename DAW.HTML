DAW

DAW significa "Desarrollo de aplicaciones Web" específicamente, en este Grado Superior (DAW) se estudia:

Primer año de DAW:

  • Sistemas informáticos (SI)
  • Arquitectura y programario (AIP)
  • Bases de datos (BD)
  • Programación (PRO)
  • Lenguajes de marcas y sistemas de gestión de información (LMSGI)
  • Entornos de desarrollo (ED)
  • Formación y orientación laboral (FOL)

Segundo año de DAW:

  • Desarrollo web en entorno cliente (DWEC)
  • Desarrollo web en entorno servidor (DWES)
  • Despliegue de aplicaciones web (DAW)
  • Diseño de interfaces web (DIW)
  • Proyecto de desarrollo de aplicaciones web (PDAW)
  • Formación en centros de trabajo(FCT)

Es decir, el Grado Superior de DAW se centra en el desarrollo de habilidades prácticas en programación, diseño y desarrollo de aplicaciones web, así como en la creación de interfaces de usuario y aplicaciones móviles. El objetivo es formar a los estudiantes en las competencias necesarias para diseñar, programar y desplegar aplicaciones web de manera profesional.
